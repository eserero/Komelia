# Product Mission

## Problem

Comic and manga enthusiasts who self-host [Komga](https://komga.org/) — an open-source media server for digital comics, manga, and books — are limited to using Komga's web UI to browse and read their collections. The web UI lacks the polish, performance, and platform integration that native apps provide.

## Target Users

Comic and manga enthusiasts who self-host Komga and want a native, polished client application to browse and read their collections across devices.

## Solution

Komelia is a cross-platform native client for Komga servers, built with Kotlin Multiplatform and Compose Multiplatform. It targets Android, Desktop (Linux/Windows), and Web (WASM), providing a superior reading and browsing experience compared to using Komga's web UI directly. Native apps unlock platform-specific integrations, better performance, offline reading, and a more refined UX tuned for comic and manga reading.
