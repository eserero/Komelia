# Product Roadmap

## Phase 1: MVP

Core functionality for a fully usable Komga client:

- Library browsing (series, books, collections, reading lists)
- Comic/manga reader with smooth page navigation
- Reading progress tracking (sync with Komga server)
- Multi-server connection support
- Android, Desktop (Linux/Windows), and Web (WASM) targets

## Phase 2: Post-Launch

Enhanced library UI and management features:

- Advanced filtering, sorting, and custom library views
- Improved library browsing UX with richer visual layouts
- Pill-based navigation and contextual panels (e.g., "Keep Reading")
- Better metadata display and organization tools
